Tela de opções Selenium

Esta tela permite a você configurar os requisitos de alguns dos WebDrivers.

Explicação das opções de configuração

Opções de configuração
Campo Detalhes Padrão
ChromeDriver Isso permite que você selecione o local do ChromeDriver. (Nenhum)
IEDriverServer Isso permite que você selecione o local do IEDriverServer. (Nenhum)
PhantomJS binário Isso permite que você selecione o local do binário do PhantomJS. (Nenhum)

Também é possível definir os locais acima usando as propriedades de sistema do Java, caso em que as opções acima serão substituídas e assumirão esses valores.
